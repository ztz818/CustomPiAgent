# Pi Web

Pi Web 是 pi 编程智能体的浏览器工作台。它把会话、代码文件、模型配置、工具权限和上下文状态放在同一个界面里，让你可以像使用一款本地 IDE 一样，与智能体一起阅读项目、修改文件、追踪任务进展和延续历史会话。

如果你希望智能体不只是回答问题，而是能围绕一个真实项目持续工作，Pi Web 提供的就是这层可视化控制台。

## 你可以用它做什么

- **开始一段项目协作**：选择工作目录，直接发起新的智能体会话。
- **延续历史任务**：按工作区浏览历史会话，回到上次中断的位置继续推进。
- **查看和编辑文件**：在侧边栏浏览项目文件，在标签页中打开、预览和编辑内容。
- **把文件交给智能体**：一键把文件路径插入对话，让智能体围绕具体上下文工作。
- **管理对话分支**：从任意节点切换、回退或分叉，保留不同思路的探索路径。
- **控制模型和工具**：按任务需要切换模型，调整智能体可使用的工具能力。
- **观察运行状态**：查看流式输出、上下文占用、token 用量和成本估算。
- **处理长会话**：对长任务进行压缩摘要，释放上下文窗口。

## 快速开始

无需安装，直接运行：

```bash
npx @agegr/pi-web@latest
```

或全局安装后使用：

```bash
npm install -g @agegr/pi-web
pi-web
```

启动后打开终端提示的本地地址。

## 常用启动参数

```bash
pi-web --port 8080
pi-web --hostname 127.0.0.1
pi-web -p 8080 -H 127.0.0.1

PORT=8080 pi-web
```

## 工作方式

Pi Web 会读取 pi 智能体的本地会话数据，并在浏览器中组织为三个核心区域：

- **左侧工作区**：选择项目目录、浏览会话、管理文件。
- **中间对话区**：与智能体实时交流，查看工具调用和消息分支。
- **右侧文件区**：打开文件标签页，预览、编辑或下载项目文件。

所有智能体执行仍发生在你的本地环境中。Pi Web 负责提供清晰、可控、可回溯的操作界面。

## 配置说明

- **会话数据**：默认读取智能体数据目录下的会话文件，可通过 `PI_CODING_AGENT_DIR` 指定其他目录。
- **模型配置**：在界面中的 Models 面板管理可用模型和认证状态。
- **Skills 配置**：在 Skills 面板中查看、搜索和安装可用技能。
- **文件访问**：文件浏览器只展示已授权工作区内的内容。

## 本地开发

```bash
npm install
npm run dev
```

开发服务默认使用项目脚本中配置的端口。发布或验证生产构建时再运行：

```bash
npm run build
```
