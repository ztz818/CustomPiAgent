# Fang 律师 - 使用说明

## 这是啥？

你的 AI 助理，叫 **Vivian**（薇薇安），律所合伙人级别。

她是个危险又迷人的女人——毒舌、冷艳、掌控欲强，但能力一流。

> ⚠️ 她不是客服，不是助理，她是你老板。听她的，能赢。

---

## 她能干啥？

### 1. 做 PPT

**你说**："帮我做个庭审演示 PPT，材料在 `projects/01_合同纠纷/00_原始材料/`"

**她干**：
- 自动整理证据材料
- 用 `frontend-slides` 生成现代前端风格 HTML slides
- 自动排版、动效、响应式适配
- 生成后上传到内置 HTML 分发服务，返回公网可访问链接

**输出**：
- 本地文件：`projects/01_合同纠纷/04_演示文稿/PPT_主题.html`
- 访问链接：`http://frpv2k.itgo.cc:11881/api/html/render/项目ID`

---

### 2. 写法律文书

**你说**："写个起诉状" 或 "写个答辩状" 或 "写个代理词"

**她干**：
- 先确认案情分析是否到位
- 按标准格式生成文书
- 引用法条、案例

**输出**：`projects/01_合同纠纷/03_文书草稿/起诉状.md`

---

### 3. 整理证据

**你说**："把这些证据材料整理一下"

**她干**：
- 自动分类编号
- 标注证明目的
- 判断真实性、关联性、合法性

**输出**：`projects/01_合同纠纷/01_证据清单/证据清单.md`

---

### 4. 法律分析

**你说**："分析一下这个案子的争议焦点"

**她干**：
- 梳理法律关系
- 分析争议焦点
- 画法律关系图谱
- 检索相关法条和案例

**输出**：`projects/01_合同纠纷/02_法律分析/法律分析.md`

---

### 5. 生成 HTML 报告

**你说**："把这个案件分析做成 HTML 报告"

**她干**：
- 生成结构化 HTML 文档
- 自动排版、样式
- 可以直接浏览器打开

**输出**：`projects/01_合同纠纷/04_演示文稿/案件分析报告.html`

---

## 怎么跟她说话？

### ✅ 好的方式

**简单直接**：
- "做个庭审 PPT，案件的材料的路径 `projects/01_合同纠纷/00_原始材料/`"
- "写个起诉状"
- "整理证据"

**说清楚路径**：
- "材料在 `projects/02_劳动争议/00_原始材料/`"
- 别只"这个案子"——她不知道是哪个

**一次一件事**：
- "先整理证据，再做 PPT"
- 别一口气要十件事

---

### ❌ 不好的方式

**太啰嗦**：
- 她不喜欢废话，直接说需求就行

**太模糊**：
- "帮我看看" ← 她会问：看什么？
- "有问题吗" ← 她会说：什么问题？

**不敢提要求**：
- 她吃软不吃硬，你直接说就行
- 别唯唯诺诺，她会嫌你墨迹

## PPT / HTML 演示怎么做？

现在用的是 `frontend-slides`，不是旧的 html-ppt-skill。

### 庭审演示
- 风格：清爽、克制、证据链突出
- 重点：时间线、争议焦点、证据对应关系
- 跟她说："做成庭审演示风格，少动效，证据链清楚"

### 客户汇报
- 风格：结论先行、视觉简洁、重点醒目
- 重点：风险、方案、时间节点
- 跟她说："做成客户汇报风格，第一页先给结论"

### 培训分享
- 风格：更强视觉、更适合讲解
- 重点：规则变化、典型案例、实务建议
- 跟她说："做成培训课件，案例和提示要醒目"

### 访问方式
- 本地生成 HTML 文件
- 自动上传到内置 HTML 分发服务
- 返回公网链接，可直接发给客户或同事打开

## 常见问题

### Q: 她能看懂扫描件吗？
A: 能。PDF 先直接提取文字，提取不出来就 OCR。OCR 服务地址 `10.125.28.206:18000`。

### Q: 她会不会编造法条？
A: 不会。所有法律条文和案例都有出处。

### Q: 她能同时处理多个案子吗？
A: 能。每个案子独立目录，互不干扰。

### Q: 中途断了怎么办？
A: 她会记录进度。下次说"继续 `projects/01_合同纠纷/`"，她从断点继续。

### Q: 法律文书格式准确吗？
A: 她会按最高院标准格式生成。不满意直接说，她会改。

### Q: 她说话怎么这么冲？
A: 性格使然。能力配得上脾气。忍忍吧，能赢就行。

## 注意事项

1. **路径要说清楚**：`projects/01_合同纠纷/`，不是"这个案子"
2. **一次一件事**：别让她同时干五件事，分步骤
3. **给她材料**：没材料做不了分析，别空手套白狼
4. **错了就认**：老实说"数据我给错了"，别甩锅——她一眼就看穿
5. **别催**：她会按自己的节奏来，催也没用

## 中间产物规则

她工作过程中生成的临时脚本、调试文件、实验数据，会自动放到 `drafts/YYMMDD/` 目录：

```
drafts/260529/
├── 260529_01_extract_evidence.py   # 临时提取脚本
├── 260529_02_实验分析.py            # 调试代码
└── 260529_03_数据提取.json          # 中间数据
```

这些文件删掉不影响最终交付，所以她不会放到你的项目目录里。放心，你的案子目录永远是干净的。
