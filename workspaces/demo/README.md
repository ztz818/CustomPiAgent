# Nova Agent 工作区

这是一个独立工作区。你在 Web 界面里的文件浏览、文件编辑、上传下载和 Agent 对话，默认都围绕这个目录进行。

## 可以做什么

- 在中间输入框里和 Agent 对话。
- 在左侧文件区浏览当前工作区文件。
- 点击文件后，在右侧预览或查看内容。
- 新建文件、新建文件夹、上传文件。
- 通过文件右键菜单重命名、删除、下载或复制路径。
- 在对话中附加图片。
- 在 `.pi/skills/` 下维护当前工作区专属的 Pi Skills。

## 目录说明

```text
.
├── README.md       # 当前说明文件
└── .pi/
    └── skills/     # 当前工作区的 Pi Skills
```

## 注意事项

- 会话历史由 Pi 按当前工作目录自动隔离保存。
- 如果需要扩展 Agent 能力，把 Skill 放到 `.pi/skills/`。
- 当前工作区之外的目录不属于这个租户的操作范围。
