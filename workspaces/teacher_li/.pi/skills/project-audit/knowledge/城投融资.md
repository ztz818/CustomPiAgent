# 城投融资项目专业知识库

## 适用场景
当审计对象为地方政府融资平台、城投企业融资业务时，参考本知识库。

## 核心关注点

### 1. 隐性债务风险
- 政府是否违规提供担保、承诺、兜底
- 项目是否为公益性项目（不得融资）
- 借款主体是否在隐性债务监测平台名单

### 2. 化债合规性
- 置换贷款：原债务是否真实纳入监测平台、是否足额结清
- 清欠贷款：是否在清欠名单、资金是否封闭划转

### 3. 项目真实性
- 立项是否在投资项目在线审批监管平台备案
- 建设进度与放款进度是否匹配
- 经营权转让是否在公共资源交易中心公开

### 4. 关键验证点
- 搜索"[项目名称] 招标公告"验证项目真实性
- 搜索"[借款主体] 隐性债务"查询是否涉隐
- 登录投资项目在线审批监管平台查询立项
