# 审计工作流程详细指南

本目录包含项目审计十步法的详细操作指南。

## 使用说明

- **SKILL.md** 提供快速索引和核心原则
- **本目录** 提供每一步的详细操作方法

## 十步法详细流程

### 第一步：了解项目基本情况

**读取项目文档，提取核心信息：**

1. **项目基本信息**
   - 项目名称、建设地点、建设内容
   - 总投资额、建设期、运营期
   - 项目当前阶段（立项/建设/运营）

2. **融资情况**
   - 借款主体、借款金额、贷款期限
   - 资金来源（自有资金、银行贷款、专项债等）
   - 还款来源（项目收益、财政补贴、其他）

3. **运营模式**
   - 自营/委托运营/特许经营/PPP？
   - 收益模式（使用者付费/政府付费/可行性缺口补助）

**记录信息来源**：每条信息标注 `文件名+页码/行号`

---

### 第二步：公开网站核实项目真实性

**⚠️ 强制要求：所有互联网查询必须使用 `web_search` MCP 工具**

**招投标信息查询**：
```python
web_search(query="[项目名称] 招标公告 site:ggzy.gov.cn")
```
- 验证：项目是否真实存在、建设内容是否一致、投资规模是否匹配
- 记录：搜索时间、关键词、结果来源URL

**项目立项查询**：
```python
web_search(query="[项目名称] 立项 备案 site:tzxm.gov.cn")
```
- 验证：项目是否已立项、立项信息是否与融资材料一致
- 记录：立项文号、立项时间、投资规模

**建设进度查询**：
```python
web_search(query="[项目名称] 施工进度")
web_search(query="[项目名称] 竣工验收")
```
- 验证：项目建设进度是否与放款进度匹配
- 记录：进度信息来源、发布时间

---

### 第三步：查看项目地址现状

**⚠️ 使用 `web_search` MCP 查询地图信息**

**地图影像查看**：
```python
web_search(query="[项目地址] 卫星地图")
web_search(query="[项目地址] site:tianditu.gov.cn")
```
- 验证：项目地址是否真实存在、建设进度是否与材料一致
- 记录：查询时间、地图来源、影像日期

---

### 第四步：查看项目证件真实性

**⚠️ 使用 `web_search` MCP 验证证件信息**

**核查关键证件（根据项目类型选择）：**

1. **立项文件**
   ```python
   web_search(query="[项目名称] 发改委 备案 OR 核准")
   ```
   - 查询官方网站验证真实性

2. **土地手续**
   ```python
   web_search(query="[项目名称] 不动产权证 OR 土地使用权")
   ```

3. **建设手续**
   ```python
   web_search(query="[项目名称] 建设工程规划许可证")
   web_search(query="[项目名称] 环评批复")
   ```

4. **特许经营/经营权**
   ```python
   web_search(query="[项目名称] 特许经营 site:ggzy.gov.cn")
   web_search(query="[项目名称] 经营权转让")
   ```
   - 验证：经营权是否真实、是否重复转让

**验证方法**：
- 对比证件信息与官方网站查询结果
- 检查证件是否有造假痕迹
- 记录验证时间和信息来源

---

### 第五步：重复融资核查

**⚠️ 使用 `web_search` MCP 查询融资信息**

**专项债查询**：
```python
web_search(query="[项目名称] 专项债")
web_search(query="[借款主体] 专项债券")
```
- 验证：专项债金额 + 银行贷款金额是否超过总投资

**债券发行查询**：
```python
web_search(query="[企业名称] 公司债 OR 企业债 募集说明书")
```
- 下载：募集说明书，核对项目信息

**中登网查询**：
```python
web_search(query="[项目名称] 质押 site:zhongdengwang.org.cn")
web_search(query="[企业名称] 应收账款质押")
```
- 验证：第一还款来源是否已被质押给其他债权人

---

### 第六步：核实资本金到位及来源

**资本金核查：**

1. **到位情况**
   - 查看放款档案，核实资本金是否先于贷款到位
   - 验证资本金比例是否符合监管要求

2. **来源合规性**
   - 资本金来源：自有资金/股东注资/政府注资？
   - 关注：是否存在以债抵资、抽逃资本金

3. **真实性验证**
   - 查看银行流水，验证资金真实到位
   - 关注：资金是否体外循环、是否被抽回

---

### 第七步：核实贷款流向和还款来源

**贷款流向核查：**

1. **受托支付核查**
   - 查看放款档案，核实是否按合同约定支付
   - 验证：资金是否支付给项目相关供应商/施工方

2. **资金流向追踪**
   - 查看银行流水，追踪资金去向
   - 关注：
     - 是否流向地方城投或财政体系
     - 是否用于化债
     - 是否被挪用至其他用途

3. **建设进度匹配性**
   - 对比放款进度与建设进度
   - 验证：资金使用是否与项目进度匹配

**还款来源核查：**

1. **第一还款来源**
   - 项目收益/财政补贴/其他？
   - 验证：还款来源是否真实可靠

2. **还款资金追踪**
   - 查看还款流水
   - 关注：还款资金是否来自地方城投或财政体系

---

### 第八步：项目总投与回款能力分析

**总投合理性分析：**

1. **成本测算核查**
   - 查看可研报告、审查报告
   - 对比：工程造价、设备采购价格是否合理
   ```python
   web_search(query="[项目类型] 工程造价 单价")
   web_search(query="[设备名称] 市场价格")
   ```
   - 验证：是否虚高总投

2. **收入测算核查**
   - 查看收入预测依据
   ```python
   web_search(query="[地区] [项目类型] 收费标准")
   web_search(query="[地区] [产品] 市场价格")
   ```
   - 对比：价格、产量是否符合市场水平
   - 验证：是否虚增收益

3. **现金流测算核查**
   - 查看现金流预测
   - 验证：偿债覆盖率是否合理

**回款能力分析：**

1. **收益来源分析**
   - 使用者付费/政府付费/可行性缺口补助？
   - 验证：收益来源是否可靠

2. **市场价格核查**
   ```python
   web_search(query="[地区] [项目类型] 租金 OR 房价 OR 水价")
   ```
   - 对比：项目预测价格是否合理

3. **偿债能力评估**
   - 计算：项目现金流是否足以覆盖债务本息

---

### 第九步：借款主体信用风险评估

**⚠️ 使用 `web_search` MCP 查询信用信息**

**主体信用风险分析：**

1. **区域风险**
   ```python
   web_search(query="[地区] 政府债务率")
   web_search(query="[地区] 财政收入 化债")
   ```
   - 判断：区域信用风险等级

2. **企业财务状况**
   - 查看财务报表
   - 关注：
     - 资产负债率、流动比率、速动比率
     - 货币资金是否覆盖短期有息债务
     - 其他应收款占比是否畸高
     - 在建工程资本化费用是否异常

3. **外部负面信息**
   ```python
   web_search(query="[企业名称] 债务违约 OR 失信 OR 涉诉")
   web_search(query="[企业名称] 拖欠工资 OR 拖欠账款")
   web_search(query="[企业名称] 失信被执行人")
   ```
   - 关注：
     - 债务违约
     - 涉诉信息
     - 失信被执行人
     - 拖欠工资/账款

---

### 第十步：风险识别与问题发现

**基于文档分析和互联网验证，识别风险：**

#### 10.1 项目真实性风险

**虚假项目**：
- 项目未立项或立项造假
- 项目未实际建设
- 经营权虚假或不存在

**虚增规模**：
- 虚高总投资
- 超额融资
- 重复融资

**材料造假**：
- 证件造假
- 合同造假
- 财务报表造假

#### 10.2 合规性风险

**审批手续不全**：
- 缺少必要的审批文件
- 审批文件过期或失效

**违反监管规定**：
- 违反行业监管要求
- 违反内部制度流程

#### 10.3 资金风险

**资本金不实**：
- 资本金未到位
- 以债抵资
- 抽逃资本金

**资金被挪用**：
- 资金流向与合同约定不符
- 资金被政府统筹使用
- 资金用于化债

#### 10.4 信用风险

**区域风险**：
- 弱资质区县
- 债务率高、财政收入低

**企业风险**：
- 财务状况恶化
- 偿债能力不足
- 外部负面信息

#### 10.5 特定行业风险（根据项目类型判断）

**城投融资项目**：
- 政府违规承诺（保底收益、缺口补助、财政兜底）
- 公益性项目融资（公园、绿地、政府大楼等）
- 涉隐平台新增融资

**特许经营/经营权项目**：
- 经营权虚假或不合理
- 项目无实际经营收入
- 实质为政府付费项目

**PPP项目**：
- 包装无收益项目为PPP
- 违规打捆公益性项目
- 政府保底承诺

**高标准农田/设施农业**：
- 土地流转不真实
- 面积虚增
- 产出数据失真

**停车场/经营权转让**：
- 重复转让
- 虚增停车位数量
- 配合政府虚增财政收入

## 注意事项

1. **灵活应用**：根据项目实际情况调整审计重点，不必机械执行所有步骤
2. **证据为王**：每个结论都要有充分的证据支撑
3. **专业判断**：结合行业知识和经验，做出合理判断
4. **持续更新**：审计方法和工具会不断演进，保持学习
